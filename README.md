# Tiles Gallery

## Purpose

Tiles Gallery is a modern tile showcase website where users can explore different tile designs, search tiles by title, view detailed tile information, and manage a user profile after login. The project is built with a clean responsive layout for desktop, tablet, and mobile devices.

## Project Theme

A website to showcase a tile gallery.

## Key Features

1. Responsive tile gallery design with Home, All Tiles, Tile Details, Login, Register, My Profile, and Update Profile pages.
2. Search functionality on the All Tiles page to find tiles by title.
3. Private route demo system for Tile Details, My Profile, and Update Profile pages.
4. User login/logout demo system using localStorage.
5. Profile update feature for changing user name and image URL.
6. Custom Navbar and Footer with quick links and contact information.
7. 404 Not Found page implementation.
8. JSON-based tile data loading from the public folder.

### Home Page

The Home page includes a banner section with the heading "Discover Your Perfect Aesthetic", a Browse Now button, a scrolling marquee, and a Featured Tiles section showing the top tile items.

### All Tiles Page

The All Tiles page displays all tile cards and includes a search input to filter tiles by title.

### Tile Details Page

The Tile Details page shows a large tile preview, title, creator, material, dimensions, price, stock status, style description, and tags.

### Authentication Pages

The project includes Login and Register pages with email/password form fields and a Google login demo button.

### My Profile Page

The My Profile page displays logged-in user information including name, email, and profile image.

### Update Profile Page

The Update Profile page allows users to update their name and image URL.
